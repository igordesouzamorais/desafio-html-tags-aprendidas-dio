O que é o HTML

A linguagem de marcação de hipertexto ou HTML é um conjunto de TAGS que juntas compõe a estrutura de uma página WEB.

Cada uma dessas TAGS tem um significado, onde juntas formam uma página WEB.

Apesar de não ser uma linguagem de programação o HTML está totalmente presente em todas as aplicações WEB.

"Hipertexto" refere-se aos links que conectam páginas da Web entre si, seja dentro de um único site ou entre sites. Links são um aspecto fundamental da web. Ao carregar conteúdo na Internet e vinculá-lo a páginas criadas por outras pessoas, você se torna um participante ativo na world wide web.

TAGS no HTML

As TAGS são extremamente importantes no HTML, elas delimitam o conteúdo do que está sendo aprensentado na página, elas são descritas dentro dos caracteres menor < e maior > e cada uma representa algo.

Um exemplo é a TAG p que representa um paragrafo dentro da página, que é um conteúdo de texto sobre o que está sendo falado.

Exemplos de TAGS:

  1. h1
  2. p
  3. a
  4. strong
  5. small

Atributos no HTML

Dentro das TAGS HTML existem os atributos, que como o nome diz são atributos que modificam a TAG, existem vários atributos, alguns globais que servem para qualquer TAG, como por exemplo o atributo id, também existe os atributos que são específicos, como por exemplo o atributo href da tag a

Os atributos são escrito dentro da abertura da TAG e seu valor é colocado após o simbolo "=".

TAGS H e como usar

As TAGS H são responsáveis pelos titulos da página, elas vão desde H1 até o H6, onde cada número desse corresponde a um peso dentro da página, sendo o H1 o título mais importante e o H6 o menos importante.

Essas TAGS são renderizadas diferentes na página mas podem ser modificadas livremente pelo CSS mas o que importa ao utiliza-las é o peso de cada uma, então é importante analisar isso na hora de usa-las.

TAG a e como usar

A TAG a é usada para redirecionar um link para uma pagina especifica fora do site ou para o proprio site, também é possível como ela que você já para locais especificos dentro da propria pagina com ajuda do atributo id.

É obrigatorio no uso da TAG a que seja passado um texto, caso contrário ela não será exibida na página e não vai funcionar, utilizando o atributo href é possível informar para onde esse link será enviado.
